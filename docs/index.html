Quickstart & Demo

Just copy the below code and save it in an html file. Then open it using a browser 

  <!doctype html>
  <html>
  <head>
    <script src="https://wayne-shih.github.io/OpenAPI-PDF-Generator/rapipdf-min.js"></script>
  </head>
  <body>
    <rapi-pdf
      style = "width:700px; height:40px; font-size:18px;" 
      spec-url = "https://wayne-shih.github.io/OpenAPI-PDF-Generator/examples/specs/petstore.json"
      button-bg = "#b44646" 
    > </rapi-pdf>
  </body> 
  </html>
